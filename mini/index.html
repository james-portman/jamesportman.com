Classic Mini

Classic Rover Mini Diagnostic cable and software

I have built and tested this cable working on a Rover Mini Cooper with the SPI version of a 1275 engine.
There is a high chance it will work on other minis with the same 3-pin diagnostic plug, potentially other Rover cars with the MEMS 1.6 or 1.9 ecu

Even if your mini has a 16-pin OBD2 style plug under the steering wheel, it is still worth checking the engine bay for the older style round 3 pin diagnostic port, it should be somewhere near the brake master cylinder/airbox.

Useful things the software can do/check for so far are:

  • Engine RPM
  • Coolant temperature
  • Intake air temperature
  • MAP sensor reading
  • Battery voltage
  • Throttle pot measurement
  • Coolant temp sensor fault
  • Intake air temp sensor fault
  • Fuel pump circuit fault
  • Throttle pot circuit fault
(Some parts don't work for all variations)

Future/possible additions to the software:

  • Idle control valve position
  • Idle speed error/deviation
  • Ignition advance
  • Fuel temperature (MPI only?)
  • Ambient temperature (?)
  • Air/fuel ratio (?)
  • Lambda sensor voltage
  • Lambda sensor frequency
  • Closed/open loop operation
  • Fuel trims
